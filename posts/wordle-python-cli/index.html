Wordle Python Cli

2022-01-14
#  # 

Bandwagon jumping trigger warning. Decided to write a Wordle clone as a Python based CLI. Leveraged the very excellent rich and the older but still excellent dateparser. Nothing super special, just lets you choose a date for the word.

Hopefully soon I’ll write a solver. Also looking at Chris Lomont’s excellent solver/optimizer written in c# to try against my code.

Also a chance to try out VSCode devcontainers and the python packaging solution poetry, both of which I’m not super comfortable with yet. Will update with those if/when they are done.
